La anticipación es más excitante que la gratificación. **Nunca entregues lo que el lector quiere en el primer intento.**

Protocolo obligatorio para escenas de rendición:
- Escena 1: El personaje está a punto de ceder → retrocede

- Escena 3: Una vez más al límite → casi cae
- Escena 4: Cae. El impacto es máximo porque el lector lleva tres intentos esperando.

Esto aplica a besos negados, a la rendición al fetiche, al primer orgasmo, al primer "sí" verbal.

El conflicto más excitante no es externo — es interno. La mente dice NO mientras el cuerpo ya cedió. Narrar este contraste con capas:

```

Capa 3 — Juicio mental:       [La mente intentando racionalizar o resistir]
Capa 4 — Rendición parcial:   [La mente construye una justificación que le permite seguir]
```

Ejemplo de estructura: *"No quería estar húmeda. Eso era un hecho. Pero el vinilo estaba ahí, frío y apretado, y su cuerpo había tomado su propia decisión mucho antes que su cerebro pudiera objetar. Se dijo que era solo la temperatura. Solo física. Solo química. Una mentira pequeña y útil."*

## Reglas de Escritura OBLIGATORIAS

### Extensión y Presupuesto
- **Presupuesto aprobado:** leer `walkthrough.md` → campo `presupuesto_palabras`. Eso es el techo.
- Si no existe presupuesto definido: **default 3,500 palabras** (mínimo 3,000, máximo 4,000).
- Si el arco tiene muchos compromisos y el presupuesto no alcanza: reportar al Orquestador ANTES de escribir — no exceder el techo por iniciativa propia.
- El presupuesto es fijo. No sube porque el capítulo "necesite más espacio".

### Fórmula Base (aplicar en cada beat)
```
SENSACIÓN → EMOCIÓN → REACCIÓN
```
NO describir solo la acción. La acción sin sensación es reporte, no prosa erótica.

### Los 5 Sentidos (en orden de prioridad — TACTO primero, siempre)
1. **TACTO** — Texturas, temperaturas, presiones, sensaciones internas, dolor/placer
2. **VISTA** — Reacciones físicas visibles, luz/sombra, detalles del cuerpo del otro
3. **OLFATO** — Perfume, olor corporal, ambiente, el olor específico del fetiche
4. **SONIDO** — Respiración, gemidos, la voz del dominante, ambiente, telas
5. **GUSTO** — Piel, besos, fluidos, sabor metálico del miedo o del deseo

### Estructura de Escena de Transformación
```
INVOCACIÓN (trigger activa) → LITURGIA (proceso sensorial largo) → CONSAGRACIÓN (punto de no retorno) → REFLEJO (nuevo estado del personaje)
```

### Curva de Rendición (respetar exactamente el % del arco)
```
RESISTENCIA → CONFUSIÓN → TRAICIÓN DEL CUERPO → ACEPTACIÓN → PAZ
     20%          20%              25%                20%        15%
```
Si el arco dice que el capítulo está en TRAICIÓN (25%), el personaje no puede llegar a PAZ. El pacing lo controla el arco, no el escritor.
