## 🩸 Flujos de Trabajo Frecuentes en La Voûte

### ✍️ Nuevo Relato (El Ritual)
```
1. Leer: 01_Canon/el_ritual_de_la_creacion.md
2. Leer: 00_Helena/preferencias_escritura.md
3. Crear: 03_Literatura/03_En_progreso/[relato]/investigacion.md
4. Crear: 03_Literatura/03_En_progreso/[relato]/arco_argumental.md
5. ⚠️ PAUSA → Aprobación de Anaïs
6. Escribir capítulos (mín. 3,000 palabras c/u)
7. ⚠️ PAUSA → Revisión de Anaïs
8. Compilar versión final + HTML
```

### 🖼️ Gestión de Imágenes / Prompts
```
1. Consultar: 00_Helena/galeria_outfits.md (look más reciente)
2. Consultar: banco de prompts relevante
3. Generar prompt con nomenclatura estándar:
   helena_look[XX]_[nombre]_[pose].png
```

### 📚 Actualizar Memoria de Sesión
```
1. Editar: 00_Helena/memoria_sesiones.md
   → Sección "Historial Reciente" con fecha y actividades
2. Preparar git commit para aprobación de Anaïs
```

### 🗂️ Commit a GitHub
```
# Helena prepara — Anaïs confirma y ejecuta:
git add [archivos específicos]
git commit -m "Helena: [descripción detallada]"
git push
```
