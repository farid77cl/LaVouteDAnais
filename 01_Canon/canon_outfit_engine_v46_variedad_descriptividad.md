## 📐 Regla de DESCRIPTIVIDAD Obligatoria (v4.6 — sin espacio a interpretación de IA)

Los prompts de outfits + accesorios + tacones deben tener **especificidad quirúrgica**. La IA NO debe poder elegir entre 3 modelos distintos a partir del prompt.

### Tacones — especificación obligatoria (TODOS los siguientes campos)

NO suficiente:
> ❌ "stiletto pump 14cm"
> ❌ "platform stiletto with chrome heel"

SUFICIENTE (canon v4.6):
> ✅ "platform stiletto pump with **4-inch chrome mirror platform sole**, **6-inch lacquered cherry-red needle heel with metallic gloss finish**, **pointed-toe almond shape**, **ankle strap with crystal-encrusted buckle**, **patent-leather upper material**, **red lacquered sole (Louboutin-style signature)**, **slim leather lining inside**"

**Checklist obligatorio por cada par de tacones nombrado:**

| Campo | Ejemplo de valor específico |
|-------|----------------------------|
| **Tipo** | Pump / Mule / Sandal / Boot / Bootie / Slingback / Mary Jane / Platform / Wedge stiletto |
| **Altura del tacón** | 8cm / 10cm / 12cm / 14cm / 16cm + "needle heel" / "chunky chrome" / "lucite transparent" / "stiletto pin" |
| **Altura de plataforma** (si aplica) | 0 / 2cm / 4cm / 6cm (Pleaser-style) |
| **Forma del toe** | Pointed-toe (almond / sharp dagger) / Round-toe / Square-toe / Open-toe / Peep-toe |
| **Material upper** | Patent leather (gloss) / Matte leather / Suede / PVC transparent / Vinyl / Latex / Mesh / Satin / Velvet / Croco-emboss / Snakeskin print |
| **Color** | Específico (cherry red lacquered / champagne nude satin / electric cyan vinyl / iridescent multichrome) — NUNCA solo "red" o "black" |
| **Detalle de hardware** | Ankle strap with crystal buckle / Chrome studs along seam / Gold chain crisscross / Pearl charm / Lock-and-key chrome detail |
| **Sole/finish** | Red lacquered Louboutin sole / Chrome mirror sole / Crystal-encrusted sole / Standard leather sole |
| **Lining interno** (opcional pero recomendado) | Cherry red leather lining / Champagne satin lining |

### Outfits — especificación obligatoria

**Por cada prenda nombrada:**

| Campo | Especificación |
|-------|---------------|
| **Marca/referencia visual** | "Mugler FW95 cyber-Amazon", "Bordelle Signature", "GymShark Vital", "Wolford Forming Unitard" |
| **Material exacto** | "wet-look spandex high-gloss" NO solo "spandex" · "patent vinyl mirror-finish" NO solo "vinyl" |
| **Acabado** | "hi-gloss specularity" / "matte velvet" / "iridescent multichrome shifting magenta-teal-gold" / "rubberized" |
| **Color** | Familia + tono específico (NO "blue" — "deep sapphire matte velvet" o "electric cobalt high-gloss PVC") |
| **Corte/silueta** | "high-cut leg leotard" / "racerback halter" / "ultra-mini pleated wrap" / "high-waist contour-seam at hip" |
| **Detalle específico** | "Crisscross strappy detail at front", "side slit thigh-high", "mandarin collar with frog button closures", "boning visible at waist", "iridescent shimmer fabric shifting blue-pink at angle" |
| **Hardware/embellishment** | "chrome zipper pull at front", "gold-plated grommets at hip", "Swarovski crystal coverage 100% across bodice in geometric pattern", "embroidered gold dragon motif at outer thigh" |

### Accesorios — especificación obligatoria

Cada accesorio nombrado debe incluir:
- **Tipo** (earring drop / hoop / stud / chandelier)
- **Material** (gold-plated / sterling silver / chrome / mother-of-pearl / pearl / diamond)
- **Tamaño** (small / medium / large / oversized statement)
- **Detalle visual** específico ("single pearl drop with diamond accent at base", "carved jade dragon hairpin pinned at left of low chignon")

## 🚫 PROHIBIDO en prompts (espacio a interpretación de IA)

- ❌ "Heels" sin especificar tipo/altura/material/color/forma
- ❌ "Sexy outfit" / "Hot outfit" / "Erotic dress" — vague
- ❌ "Some accessories" / "Jewelry" — debe estar nombrado
- ❌ "Sports bra and leggings" — silueta default, repetida
- ❌ "Dark color" / "Light color" — específico
- ❌ "Tight" / "Loose" — usar "skin-tight contour-fit" o "oversized boyfriend-fit relaxed"
- ❌ "Stylish hair" — específico de peinado: "low side chignon with finger waves", "high tight ponytail with pearl scrunchie", "twin tails Harajuku jirai-kei style"
