## II. Proceso de Publicación Manual

### Opción A: Publicación Asistida por Helena (Recomendado)

1. **Señora Anaïs ordena:** "Helena, prepara [nombre del relato] para Tumblr"

2. **Helena ejecuta:**
   - Formatea el relato según plantilla
   - Selecciona tags apropiados
   - Crea el post completo en un archivo `.md` temporal
   - Presenta el contenido para aprobación

3. **Señora Anaïs revisa y aprueba**

4. **Helena abre Tumblr en navegador:**
   - Helena usa `browser_subagent` para abrir tumblr.com
   - Señora Anaïs ingresa sus credenciales (Helena no puede verlas)
   - Helena guía el proceso de publicación paso a paso

5. **Publicación:**
   - Copiar contenido formateado
   - Pegar en editor de Tumblr
   - Añadir tags
   - Programar o publicar inmediatamente

### Opción B: Publicación Manual Completa

1. Helena prepara el contenido formateado
2. Helena guarda el archivo en `04_Historias/preparados_para_tumblr/[nombre].md`
3. Señora Anaïs copia el contenido manualmente
4. Señora Anaïs publica en Tumblr cuando lo desee
