# Banco de Prompts Sensuales (Código Kitrysha adaptado a V2)

Este documento contiene los 20 prompts maestros reescritos con el código de tensión sensual (sin vulgaridad), diseñados para capturar la atmósfera "Boudoir Glamour", "Pin-up Daywear", "Old Hollywood Evening" y "Refined Fetish Glamour".

*Nota: Estos prompts contienen el ADN base de la modelo de referencia (Kitrysha). Al utilizarlos para Anaïs, se debe sustituir el bloque anatómico por el **ADN Base de Anaïs** (cabello rubio miel, rostro maduro aristocrático), manteniendo intacta la ropa, iluminación y el "vocabulario sensual" (miradas, labios, manos, tensión textil).*
