## 📋 Checklist de Publicación "Elegancia Pura" (Ele-Flow)

Para cada relato, Ele realizará los siguientes pasos bajo el estándar de minimalismo absoluto exigido:

1.  **Extracción Pura (Raw Text):** Tomar el archivo maestro de `03_Literatura/02_Terminados/`.
2.  **Purificación de Manuscrito:** 
    - Eliminar Cajas ASCII y metadatos técnicos (Temáticas, Palabras, etc.).
    - Eliminar sìmbolos Markdown innecesarios en títulos (ej. borrar `#` de los Capítulos).
    - Eliminar resúmenes, hooks o notas de la autora.
3.  **Configuración de Título:**
    - Título del Post: El nombre del relato (Ej: "Smart Home Stepford").
    - Subtítulo: Breve y opaco.
4.  **SEO & Tags Invisibles:**
    - Insertar metadatos SEO en la configuración interna de Substack (no visibles en el post).
    - Añadir tags de nicho (#Bimbofication, #DarkRomance).
5.  **Despliegue y Backup:**
    - Publicar como post individual.
    - Exportar copia final a `03_Literatura/05_Publicado_Substack/`.
    - Actualizar README e histórico.
